# Mission Improbable example sprites

This is a test project for `px`, for a game idea I had last year: a remake of Mission Impossible (from the 1980s). I want to use it as a use case for the tool chain.

## Files 

- mi.palette.md (base colours)
- platforms.shape.md (example platform parts)
- lifts.shape.md (example elevator parts)
- fills.shape.md (background fills, for things like rocks)
- objects.shape.md (map objects, like cabinets, chairs, etc.)
- npcs.shape.md (basic robots)
- player.shape.md (player shapes)
- level-example.map.md (a test example level)
- test.target.md
